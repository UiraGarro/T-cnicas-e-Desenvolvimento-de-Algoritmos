Estruturas Condicionais

° Permite a escolha do grupo de ações a ser executado quando determina condição é ou não satisfeita.

exemplo:

if(<condição>){
    bloco de instruções1;
}
else{
    bloco de instruções2;
}

exemplo2:

int main(){

    float nota;

printf("Digite sua media:\n");
scanf("%f", &nota);

if (nota>=7){
    printf("aprovado por media\n");
}
else{
    printf("final\n");
    }
}

exemplo3:

int main(){
    int num;

    printf("digite um numero:\n");
    scanf("%d", &num);

    if(num%2 == 0){
        printf("numero par\n", num);
    
    } else {
        printf("numero impar\n", num);
    return 0
    }
}
